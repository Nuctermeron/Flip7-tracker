# Flip 7 – Tracker (Vite + React + Tailwind)

Gotowy szkielet pod lokalne uruchomienie i deploy na Vercel.

## Lokalnie
```bash
npm install
npm run dev
# otwórz adres z konsoli, np. http://localhost:5173/
```

## Build
```bash
npm run build
npm run preview
```

## Deploy na Vercel
- Podłącz repo do Vercela
- Framework: **Vite**
- Build command: `vite build`
- Output dir: `dist`
